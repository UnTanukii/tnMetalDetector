# Metal Detector FiveM (Lua)

A simple metal detector script for FiveM

## Features

* Easy and complete configuration
* Multiple configurable zones with different loot tables
* Progressive beeping system
* Random treasure spawning
* Chance-based loot system
* Progress bar support (external but easily customizable)
* Optimized

## Installation

1. Place the `tnMetalDetector` folder in your server resources
2. Add `start tnMetalDetector` to your `server.cfg`

## Usage

* Use the item to activate the metal detector
* Move around within configured zones to detect buried items
* Beeping frequency increases as you get closer to a treasure
* Press E to dig up the treasure when detection is successful

## Dependencies

* **ESX** *(framework can be easily changed)*
* *Progress bar script (default: t0sic_loadingbar, optional and easily replaceable)*
